# Lumina Virtual Input Library

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![NuGet](https://img.shields.io/nuget/v/LuminaVirtualInput.svg)](https://www.nuget.org/packages/LuminaVirtualInput/)

Lumina Virtual Input Library is a powerful, low-level C# library for simulating mouse and keyboard inputs on Windows systems. It provides precise control over mouse movements and clicks, as well as keyboard input, without relying on higher-level .NET frameworks.
